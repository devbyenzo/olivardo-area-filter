## 📋 Sobre o Projeto

Olivardo Area Filter é uma ferramenta de automação desenvolvida para processar planilhas Excel e realizar a filtragem inteligente de registros com base nas áreas utilizadas no sistema Olivardo.

Seu principal objetivo é reduzir o tempo gasto com análises manuais, padronizar processos e melhorar a organização dos dados.

---

## 🚀 Funcionalidades

- Leitura de arquivos Excel (.xlsx)
- Filtragem automática de registros por área
- Processamento rápido de grandes volumes de dados
- Organização e separação de informações relevantes
- Exportação dos resultados processados
- Redução de erros manuais

---

## 🛠️ Tecnologias Utilizadas

- Python 3.10+
- Pandas
- OpenPyXL
- Tkinter (caso utilize interface gráfica)

## ⚙️ Instalação

Clone o repositório:

```bash
git clone https://github.com/seu-usuario/olivardo-area-filter.git
```

Entre na pasta do projeto:

```bash
cd olivardo-area-filter
```

Instale as dependências:

```bash
pip install -r requirements.txt
```

---

## ▶️ Como Utilizar

1. Coloque a planilha que deseja processar na pasta apropriada.
2. Execute o sistema:

```bash
python main.py
```

3. Aguarde o processamento.
4. O arquivo filtrado será gerado automaticamente.
